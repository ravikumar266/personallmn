# PersonaLLM
# 🧠 Persona LLM – AI Argument Evaluation Challenge

Persona LLM is a web-based platform where users submit structured arguments from the perspective of famous personas (like Albert Einstein), and an AI system evaluates them.

---

## 🚀 Features

- 🎭 Persona-based argument generation
- 🧠 AI-powered evaluation (Critic LLM)
- 📊 Score breakdown system
- 🏆 Leaderboard tracking
- 📁 JSON-based submission format

---

## 🛠️ Tech Stack

- **Frontend:** Next.js
- **Backend:** Next.js API Routes
- **Database:** MongoDB Atlas
- **AI Evaluation:** GROQ API
